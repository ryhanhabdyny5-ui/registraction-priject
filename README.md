# User Registration Form - Web Development Project

## Overview
This project is a simple user registration form built with pure HTML, CSS, and JavaScript.  
It includes real-time validation for username, full name, email, and password fields.

## Features
- Pixel-perfect layout matching Figma design
- Real-time validation with error messages
- Password validation:
  - Minimum 8 characters
  - Must include at least one number or symbol
  - Cannot contain user's name or email
- Submit button disabled until form is valid
- Success message on submission
- Console log of form data

## How to Run Locally
1. Clone the repository
2. Open index.html in any modern browser
3. Fill out the form and check validation
